# Hey these are my notes, todo and more

## Vim keybindings

new window:
    horizontal: Ctrl w s
    vertical: Ctrl w v
    undo: u - :undolist
    redo: Ctrl r
    :Ex: Shift l

## Tmux keybindings

    Ctrl + b + % to split the current pane vertically.
    Ctrl + b + " to split the current pane horizontally.
    Ctrl + b + x to close the current pane.

## Bash tricks

    pass fzf result to any program: mdless "$(fzf)"
